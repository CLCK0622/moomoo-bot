# ❌ 固定止损阈值 -$2,000

### 现在（动态计算）
```python
# ✅ 动态止损阈值 = 单只股票预算的1%
# 1. 计算单只股票预算
fixed_budget = (total_assets / 1.2) / approved_count

# 2. 计算止损阈值（预算的1%）
stop_loss_threshold = -1 * (fixed_budget * 0.01)

# 3. 应用止损
if today_pnl < stop_loss_threshold:
    print(f"🚨 [{sym}] MELT DOWN! PnL ${today_pnl:.2f} < ${stop_loss_threshold:.2f}. FORCE CLOSING.")
```

---

## 📊 计算公式

### 单只股票预算
```
单只预算 = (总资产 / 1.2) / 股票数量
```

### 止损阈值
```
止损阈值 = -1 × 单只预算 × 1%
```

**示例**（当前账户）:
```
总资产:     $999,289.38
股票数量:   5只
单只预算:   $166,548.23
止损阈值:   -$1,665.48  (预算的1%)
```

|------|----------|----------|------|
| 旧方式（固定） | -$2,000 | 固定值 | ❌ 不随预算调整 |

---

## ✅ 优势

1. **动态适配** 
   - 账户变大 → 止损阈值增加
   - 账户变小 → 止损阈值减小

2. **风险一致**
   - 固定比例1%，所有情况下风险暴露一致
   - 不会因为资产规模变化而承担不成比例的风险

3. **精准控制**
   - 每只股票的风险暴露相同
   - 符合"单只预算"的设计理念

4. **透明计算**
   - 公式清晰，易于理解和验证
   - 启动时打印具体数值

## 💡 补充说明

### 为什么是预算的1%？

1. **符合止损策略**: 之前的交易策略中硬止损是成本的-1%
2. **风险对称**: 单笔交易的硬止损是-1%，全天总止损也是-1%
3. **灵活调整**: 如果需要更严格/宽松，可以调整比例（0.5%或2%）

### 如何修改比例？

修改 `monitor.py` 第117行：
```python
# 改为0.5%（更严格）
stop_loss_threshold = -1 * (fixed_budget * 0.005)

# 改为2%（更宽松）
stop_loss_threshold = -1 * (fixed_budget * 0.02)
```
